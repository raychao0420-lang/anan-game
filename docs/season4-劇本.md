## 一、世界觀與開場

天空三部曲結束後的某個早晨，一隻灰白色的貓頭鷹羽毛夾著一封燙金信，落在安安窗前：

> 「安安同學：恭喜你偵破十二星座與太陽系連環大案。
> 茲邀請你進入**邏輯偵探學院**就讀，一級一級考取偵探執照。
> 修滿十二枚金印，即可成為——**名偵探安安**。
> —— 邏輯偵探學院 院長 福爾摩熊」

小星🌟、小月🌙、小冥🌑 全體寵物送安安上學（開場溫馨小劇場），小星照例賴著當跟班助手。
學院是一座長滿常春藤的古堡：有審訊室、圖形密室、說謊者之島訓練場、鐘樓、
圖書館禁區、後山回聲階梯……每集＝一堂課＝一種邏輯工具＋一樁真實小案件。

**收集物：12 枚「級別金印」🔏（collType `seal`，LICENSE_BOARD）**——
每破一案，院長就在安安的空白執照上蓋一枚金印；蓋滿 12 枚，執照生效、畢業領證。
（比照 S1 碎片／S2 徽章／S3 寶石通用化；小孩超愛蓋章的儀式感。）

## 三、裏層主線：等了四十年的搭檔

（品牌公式第四彈：走失🌟→迷路🌙→被遺忘🌑→**等待🦉**）

- 創辦人**阿基教授**年輕時與貓頭鷹搭檔**小Q**環遊世界解謎，回來創立學院。
- 教授年紀大了，回山上老家休養。臨走前留下傳說中的「**最後一題**」，
  告訴小Q：「等有孩子解開它，就是你的新搭檔。」
- 小Q 不肯接受離別，40 年來守著學院鐘樓，每年偷偷出題想找「像教授一樣的孩子」，
  卻沒有人解得開。教授的最後一封信，牠拆了又摺回去幾百次，**始終不敢讀完最後一段**。
- **每集固定痕跡：一根灰白羽毛🪶＋一張「問號卡」❓**（謎語小卡，12 張卡背面
  可拼成一幅熱氣球圖）。比照 S3 冰腳印，clueLabel＝灰白羽毛🪶。
- 教室永遠多一個沒人坐的「**第 13 號座位**」，有時上面會出現寫好的作業——
  小Q 旁聽了 40 年的課。
- 信的最後一段（EP11 才讀完）：
  > 「小Q，謝謝你陪我一輩子。別再等我了——
  > 去找下一個眼睛發亮的孩子，**換你當前輩了**。 —— 阿基」
- 終章：安安解開「最後一題」→ 畢業典禮上**熱氣球降落，阿基教授回來了**
  （巡巡教官一直和教授通信安排這一天；典典是教授老同學）。
  教授親手頒發名偵探執照；小Q 正式成為安安的搭檔，
  胸前問號羽毛「?」翻成驚嘆號「!」——所有謎題都有答案，等待也是。
  教授金句：「執照不是證明你多聰明，是提醒你：把想不通的事想通，把難過的人變笑。」

### EP1《入學考的密碼門》🚪 — 規律數列入門（等差）
- **邏輯工具**：找規則、下一個數（+2、+5、+10、遞減、找第 N 個）。
- **案情**：入學考當天，通往考場的七道密碼門的密碼牌全不見了！新生們被鎖在門外。
  安安沿路用數列規律推出每道門的密碼。
- **答案**：9 / 20 / 50 / [45分支：左「光之樓梯」右「藤蔓滑梯」] / 55 / 70 / 30 / 14
- **真兇＋暖心反轉**：小刺蝟同學**棘棘**——太怕自己考不上，前一晚拿走密碼牌想偷偷先練習。
  院長：「承認錯誤需要的勇氣，比解開密碼更像偵探。」棘棘成為安安的第一個同學。
- **彩蛋**：偵探始祖福爾摩斯的名言「排除所有不可能，剩下的就是真相」。
- **裏層**：考場座位表多出一個沒人的「第 13 號座位」，桌上一根灰白羽毛🪶＋問號卡#1：
  「歡迎你。我出的題，40 年來還沒有人解開過。」
- **金印**：見習偵探印。reward 450。

### EP2《真假審訊室》⚖️ — 幾句是真的
- **邏輯工具**：對照事實數真話、抓出說謊的人（答案全是數字：幾句真／第幾號說謊）。
- **案情**：福利社最後一個布丁不見了！三位同學各說三句話，安安用「事實對照法」
  一句一句驗證。
- **答案**：2 / 1 / 3 / [2分支] / 0（整組全是假話，驚喜點）/ 3 / 1 / 7（布丁在 7 號置物櫃）
- **真兇＋暖心反轉**：倉鼠**豆豆**——把布丁藏起來要留給生病請假的同學，
  卻謊稱「我沒進過福利社」。教訓（詳細教）：好意也不該說謊，說實話大家會一起幫你。
  結局：全班湊零用錢買一整盒布丁去探病。
- **彩蛋**：測謊機真的存在，但最厲害的測謊工具是「前後對不對得上」。
- **裏層**：審訊室黑板出現一道沒人出的謎語＋問號卡#2：「說謊的人，記性要很好喔。」
- **金印**：真話之眼印。reward 490。

### EP3《圖形密室的循環規律》🚩 — 週期與循環
- **邏輯工具**：週期（每 n 個一組）、用除法餘數判斷「第 N 個是什麼」、循環計數。
- **案情**：「創辦人紀念日」前夕，禮堂的傳統彩旗被重新排過，順序全亂——
  而旗序其實藏著創辦人設下的規律密碼！
- **答案**：3 / 3 / 2 / [4分支] / 10 / 6 / 4 / 18（密碼＝第 5 組旗的編號和）
- **真兇＋暖心反轉**：布置股長孔雀**彩彩**——牠發現舊旗褪色，想全部換新讓紀念日更漂亮，
  卻不小心抄錯一段順序、破壞了密碼。安安用規律還原；新旗照舊序掛上，又新又對。
  「傳統不是不能變漂亮，但要先讀懂它為什麼這樣排。」
- **彩蛋**：紅綠燈、星期、月亮圓缺——生活裡到處都是循環規律。
- **裏層**：還原的旗序密碼指向圖書館的某個書架號。問號卡#3：「答案在書裡睡覺。」
- **金印**：循環之輪印。reward 530。

### EP4《說謊者之島》🏝️ — 真話族與假話族（騎士惡棍簡化版）
- **邏輯工具**：誠實帽（只說真話）／搗蛋帽（只說假話）、反話翻譯、互相指認。
- **案情**：校外教學到湖中訓練島，回程船票全不見了！島民有的戴誠實帽、有的戴搗蛋帽，
  安安要聽懂「反話」找回船票。
- **答案**：1 / 2 / 8 / [3分支] / 2 / 3 / 2 / 1（船票藏在 1 號燈塔）
- **真兇＋暖心反轉**：擺渡人河狸**槳槳**——湖上起大霧開船危險，牠收起船票不讓大家上船，
  卻只會用搗蛋帽規則說反話提醒，沒人聽懂。「有些人表達關心的方式不一樣，
  偵探要聽懂別人**真正想說的話**。」霧散後大家平安回家。
- **彩蛋**：「說謊者悖論」小故事——「我這句話是假的」到底是真是假？（簡化版，激發思考）
- **裏層**：燈塔頂貼滿歷屆畢業合照，每張都有同一隻貓頭鷹——但**最近幾年的合照裡，
  貓頭鷹不見了**。問號卡#4：「照片會舊，等的人不會。」
- **金印**：辨真之耳印。reward 570。

### EP5《雙層規律的鐘樓》🔔 — 等差進階·隔項雙層（難度轉陡）
- **邏輯工具**：交替規律（+1+2交替）、遞增差（+1+2+3…）、隔項雙數列、倒推。
- **案情**：學院鐘樓的鐘聲全亂了，三點敲七下、七點敲兩下！鐘的齒輪刻著雙層數列，
  排錯一格全樓走音。
- **答案**：10 / 17 / 4 / [16分支] / 20 / 8 / 5 / 21（鐘聲總碼＝1+2+…+6，三角數彩蛋）
- **真兇＋暖心反轉**：敲鐘人老山羊**噹噹**——眼睛花了看不清齒輪刻度，硬撐著不說，
  怕一說就要被退休。全學院幫他配了眼鏡，安安兼任「報數小助手」。
  「需要幫忙不丟臉，說出來才是勇敢。」
- **彩蛋**：1+2+3+…排成三角形＝「三角數」，保齡球瓶就是 10！
- **裏層**：鐘樓最高層有間小圓窗房間——桌上的茶**還是溫的**，窗邊一根羽毛。
  有「人」住在這裡！問號卡#5：「你們越來越近了。」
- **金印**：雙層之眼印。reward 610。

### EP6《校慶園遊會小案簿》🎡 — 喘息集·混合輕鬆三連小案

- **案情**：校慶園遊會三連小案：撈金魚攤的數目對不上／小學弟走失了（用規律找路線）／
  獎品券多出一疊。
- **答案**：12 / 3 / 2 / [15分支] / 4 / 1 / 9 / 30

  看著熱鬧的人群，眨了眨，熄了。問號卡#6：「熱鬧真好。**從這裡看了四十年。**」

### EP7《如果就的連鎖密室》🗝️ — 條件句（如果 A 那麼 B）
- **邏輯工具**：條件句、連鎖推理（A→B→C）、簡化逆否（鐘沒響，所以不是晴天）、找出錯的一步。
- **案情**：圖書館深夜自動上鎖，典典被鎖在門外！開門要破解創辦人設的「連鎖鎖」：
  「如果紅燈亮，那麼轉 3 圈；如果轉了 3 圈，那麼按 2 次……」
- **答案**：2 / 6 / 4 / [8分支] / 5 / 9 / 3（第 3 步錯了）/ 12（開門總碼）
- **真相＋懸念**：沒有兇手——是典典每年創辦人生日那天會啟動機關整理「禁區書架」，
  今年忘了關。禁區書架上全是**創辦人的解謎筆記**，典典替「某位老朋友」保管。
  牠欲言又止：「還不到告訴你們的時候。」（紅鯡魚②半洗白，懸念留終章）
- **彩蛋**：電腦程式的本質就是一大串「如果…就…」——你已經在學寫程式的思考了！
- **裏層**：禁區書架最深處有本《我與小Q環遊世界》——**「小Q」這個名字首次出現**。
  問號卡#7：「那是我最快樂的日子。」
- **金印**：連鎖之鑰印。reward 650。

### EP8《數字金字塔的頂端》🔺 — 規律×運算

  只有排回正確的數字金字塔，獎盃才放得回頂端。
- **答案**：7 / 12 / 5 / [9分支] / 20 / 6 / 30 / 16（頂端獎盃編號）
- **真兇＋暖心反轉**：小松鼠**奇奇**——想爬上去拿頂端獎盃擦亮，當作送給畢業姊姊的禮物，

- **彩蛋**：金字塔底座刻著創辦人名言——「每個答案，都站在別的答案的肩膀上。」

  「我也想，再站上誰的肩膀看看。」
- **金印**：金字塔之頂印。reward 690。

### EP9《三真一假的畢業照》📸 — 逐句消去法（主線大高潮）
- **邏輯工具**：多句三真一假、兩兩對質、逐步消去縮小範圍。
- **案情**：檔案室歷屆畢業照少了一張——正是 EP4 發現的「貓頭鷹消失前的最後一張」！
  四位在場者各執一詞，三真一假。
- **答案**：2 / 2 / 3 / [4分支] / 1 / 1 / 38（第 38 屆，照片編號規律）/ 6（6 號抽屜）
- **真兇＋大反轉**：**巡巡教官**拿走了照片！！（貫穿紅鯡魚正面撞上）
  牠承認，卻拒絕說原因：「有些約定，比執照更重要。」把照片放回就走。
  照片裡——年輕的阿基教授，肩上站著一隻胸前有問號羽毛的貓頭鷹。
  （真相終章揭曉：教授來信說想再看看小Q 的樣子，巡巡要把照片寄上山。）
- **彩蛋**：消去法是真實刑警最常用的推理法——嫌疑人名單越劃越短，真相就浮出來。
- **裏層大進展**：深夜檔案室，安安**第一次正面遇見小Q**——灰白貓頭鷹抱著那張照片發呆，
  四目相對，丟下問號卡#9 飛走：「你就是……解開很多題的那個孩子嗎？」
- **金印**：消去之刃印。reward 730。

### EP10《回聲階梯》🪜 — 遞迴規律（前兩項相加，費波那契入門）
- **邏輯工具**：遞迴數列（後項＝前兩項相加）、缺中間項、倒推首項。
- **案情**：本集無兇手，改「闖關救援」型——安安主動去後山找小Q！
  後山「回聲階梯」每階刻著遞迴數列，踩錯就滑回起點；階梯盡頭的老橡樹樹屋，
  是當年教授和小Q 的祕密基地。小Q 傲嬌地躲著不見人，沿路留題「測試」安安。
- **答案**：8 / 13 / 18 / [21分支] / 6 / 13 / 2 / 34（樹屋門牌，全是費波那契數！）
- **結局**：安安全數解開。枝頭剪影裡，小Q 終於開口說出故事的開頭：
  「我在等一個人回來。」

- **金印**：遞迴之梯印。reward 770。

### EP11《畢業典禮的席位表》🪑 — 多條件交叉綜合（情感倒數集）
- **邏輯工具**：多條件交叉推理（A 不坐 B 旁邊／C 在第 2 排…）、排列消去、綜合。
- **案情**：籌備畢業典禮！安安負責排嘉賓席位表，一堆條件交叉限制。
  輕案：席位表上有個「第 0 排正中央」的**無名座位**——神祕嘉賓是誰？
  線索指向「一位從很遠的山上來的老先生」。（巡巡教官偷偷留的位子；
  品牌 motif「留位子」，callback S3 EP11 第 31 個位子。）

- **彩蛋**：為什麼有「第 0 排」？因為最重要的位子，要比第 1 排更前面。
- **裏層大進展（本季最淚點）**：安安把 11 張問號卡背面拼起來——是一幅**熱氣球圖**，

- **金印**：交叉之網印。reward 810。
